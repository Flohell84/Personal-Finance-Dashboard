## Frontend (Vite/React/TypeScript)

### Entwicklung
```cmd
cd new\frontend
npm install
npm run dev
```

### Build & Preview
```cmd
npm run build
npm run preview
```

### Hinweise
- Backend erwartet unter http://127.0.0.1:8000 (API unter /api)
- Admin‑Konsole sichtbar nur für is_admin=true
- CSV‑Import: UTF‑8 CSV, Spaltenzuordnung im UI
